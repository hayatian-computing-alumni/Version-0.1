Core Capabilities

Engineered for
Modern Computing

PakOS is designed with a clean architecture, real-time scheduling, and a security-first philosophy. Every subsystem is purpose-built.

⚡

Preemptive Scheduler

Custom PREEMPT_DYNAMIC scheduler with real-time priority classes and low-latency task switching optimised for modern multi-core CPUs.

🔒

Hardened Security

Stack canaries, ASLR, SMEP/SMAP enforcement, and a built-in LSM framework provide multiple layers of kernel-level protection.

🧠

Smart Memory Manager

Adaptive page allocator with transparent huge pages, NUMA-aware placement, and low-memory pressure handling out of the box.

🌐

Native Networking Stack

Rebuilt TCP/IP implementation with BPF-based packet filtering, eBPF hooks, and zero-copy socket buffers for high-throughput I/O.

🗂️

PAKFS Filesystem

An experimental journaling filesystem with copy-on-write semantics, inline data extents, and crash-consistent atomic operations.

🔧

Modular Driver Model

Hot-pluggable kernel modules, unified device tree support, and a clean driver API compatible with existing Linux kernel drivers.

// Quick Start

Clone the PakOS repository and use the provided Makefile to compile and boot the kernel image in a QEMU virtual machine.

# Clone the repository $ git clone https://github.com/pakos-kernel/pakos.git $ cd pakos # Install build dependencies $ sudo apt install build-essential libncurses-dev \ bison flex libssl-dev libelf-dev qemu-system-x86 # Configure and build $ make pakos_defconfig $ make -j$(nproc) # Boot in QEMU $ make qemu-run → Booting PakOS 0.1-alpha ... OK

// System Requirements

Minimum requirements for building and running PakOS. Development is primarily tested on Ubuntu 22.04 LTS and Fedora 39.

Arch x86_64 · arm64 (experimental) RAM 512 MB minimum · 2 GB recommended Disk 4 GB for build environment Compiler GCC 12+ or Clang 16+ Host OS Linux (Ubuntu, Fedora, Arch)

// Writing a Kernel Module

PakOS follows standard Linux module conventions. A minimal "Hello World" module looks like this:

#include <linux/module.h> #include <linux/kernel.h> #include <linux/init.h> static int __init hello_init(void) { pr_info("PakOS module loaded\n"); return 0; } static void __exit hello_exit(void) { pr_info("PakOS module removed\n"); } module_init(hello_init); module_exit(hello_exit); MODULE_LICENSE("GPL");

About the Project

Built at a University.
Open to Everyone.

PakOS is an academic open-source project born from a university operating systems research lab. Our goal is to develop a fully functional, well-documented Linux-based kernel that serves both as a research platform and a practical teaching tool for systems programming in Pakistan and beyond.

The project is 100% open source under the GPL-2.0 license. We welcome contributions from students, researchers, and engineers worldwide.
